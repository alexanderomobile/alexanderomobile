# Александр · Prompt Engineer & AI Automation

**Промпт-инженер и разработчик Telegram-ботов** с фокусом на RAG, LLM-интеграции и автоматизацию бизнес-процессов.

Проектирую цепочки «данные → контекст → ответ», упаковываю решения в понятные интерфейсы (боты, API, no-code) и довожу задачи до рабочего продукта.

---

## Компетенции

| Направление | Инструменты и практики |
|-------------|------------------------|
| **RAG и контекст-инжиниринг** | ChromaDB, Pinecone, Weaviate, чанкинг, эмбеддинги, metadata-filter, RAGAS |
| **LLM-интеграции** | OpenAI, GigaChat, Gemini; structured output (JSON/XML); multi-step reasoning |
| **Telegram-боты** | aiogram 3, python-telegram-bot, webhook + FastAPI |
| **Автоматизация** | n8n, APScheduler, парсинг, платёжные сценарии |
| **Backend** | Python, FastAPI, SQLAlchemy, MySQL, SQLite |
| **Мультимодальность** | image-to-video пайплайны, субтитры, конкурентный анализ |
